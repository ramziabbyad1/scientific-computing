Introduction

There are many free resources online about the Analytic Hierarchy Process. It is a simple, but powerful organizational tool for multi-criteria comparison analysis. In this introduction I will explain the basics of AHP, and crank out an example use case. If you create a user account, you'll be able to create and save your own AHP models.

Making Decisions

In our postmodern capitalist world, decisions control our lives. Haven't you ever walked into a grocery store looking for a jar of, say, peanut-butter, and been baffled by the sheer number of choices available? Or perhaps you're in a relationship and you can't decide whether or not to get married.

If you were to confront these decisions as a scientist, how would you go about doing that? One common technique is to make a pro-con (cost/benefit) list. The items on this list represent the criteria upon which your decision depends. Let's run with the marriage example I just mentioned:

I arranged the previous example to illustrate a point. Each row represents a criterium. The first row represents expense. The second, use of personal time. The third, points to issues involving monogamy.

How could we use this table to make an educated decision? If all of the criteria are equally important, then all we have to do is decide which item is most important on each row, the pro or the con? Then we just tally them up an make out decision:
